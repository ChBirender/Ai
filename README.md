1. create requirements.text file  
add below text:  

jupyter  
lxml  
matplotlib  
pandas  
Pillow  
scikit-learn  
numpy  
seaborn  
tensorflow  
jupyterlab  
notebook  
networkx  

in requirements.txt

2. open cmd  
python -m pip install -r requirements.txt

.\jy
